# JAYK0P - Osobní Portfolio & Webová prezentace

Vítejte v mém osobním repozitáři pro GitHub Pages! Tento web slouží jako moje hlavní portfolio a rozcestník k mým školním i soukromým projektům.

**Live verze webu:** [https://jayk0p.github.io/](https://jayk0p.github.io/)

---

## 🚀 O projektu

Tento web je vytvořen jako čistá, moderní a responzivní jednostránková prezentace s důrazem na prémiovou estetiku (dark-mode a glassmorphism design).

### Použité technologie:
*   **HTML5** – Sémantická struktura stránek.
*   **CSS3 (Vanilla)** – Vlastní CSS proměnné, responzivní layouty (Flexbox & CSS Grid), plynulé přechody, klíčové animace a skleněné efekty (`backdrop-filter`).
*   **Vanilla JavaScript** – Dynamické chování (mobilní menu, plynulé aktivní navigace při scrolování a efekt psacího stroje v Hero sekci).
*   **Font Awesome** – Moderní vektorové ikony pro dovednosti a kontakty.

## 🛠️ Jak spustit projekt lokálně

Pro lokální prohlížení webu na svém počítači:
1.  **Naklonuj repozitář** (pokud ho ještě nemáš stažený):
    ```bash
    git clone https://github.com/JAYK0P/JAYK0P.github.io.git
    ```
2.  **Otevři soubor `index.html`** přímo ve svém webovém prohlížeči (např. dvojitým kliknutím na soubor, nebo pomocí rozšíření *Live Server* ve VS Code).

## 📈 Jak nahrát změny na GitHub (Nasazení)

Kdykoliv upravíš kód lokálně a chceš, aby se změny projevily na tvém veřejném webu:

1.  **Přidej změny do gitu:**
    ```bash
    git add .
    ```
2.  **Vytvoř commit (popis změn):**
    ```bash
    git commit -m "Aktualizace portfolia: [stručný popis změn]"
    ```
3.  **Odešli kód na GitHub:**
    ```bash
    git push origin main
    ```
GitHub automaticky spustí build a do 1-2 minut se tvé změny projeví na [https://jayk0p.github.io/](https://jayk0p.github.io/).
